🎲 DevSorteio - Sorteador de Números

Um projeto simples e elegante desenvolvido para realizar sorteios de números aleatórios dentro de um intervalo definido pelo usuário. Ideal para sorteios rápidos, rifas ou tomadas de decisão.

🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias fundamentais da web:

HTML5: Estruturação dos elementos da página.

CSS3: Estilização com foco em uma interface moderna, utilizando gradientes e efeitos neon.

JavaScript: Lógica para geração de números aleatórios e manipulação do DOM.

🛠️ Funcionalidades

Intervalo Customizável: O usuário pode definir o valor mínimo e o valor máximo para o sorteio.

Interface Responsiva: O design se adapta para uma boa visualização em diferentes dispositivos.

Feedback Visual: O número sorteado aparece em destaque para facilitar a leitura.

💻 Como rodar o projeto

Clone este repositório:

Bash

git clone https://github.com/dev-alef/devsorteio.git

Navegue até a pasta do projeto:

Bash

cd devsorteio

Abra o arquivo index.html no seu navegador de preferência.

🎨 Layout

O projeto conta com um cabeçalho estilizado com números em neon (37, 9, 1, 5, 2) e um campo de entrada intuitivo. Abaixo do resultado, o usuário encontra os campos para os valores de "mínimo" e "máximo" e o botão de ação principal.
